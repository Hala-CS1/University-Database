# University Database Project

This is a group project for the Introduction to Databases course.

The project focuses on designing and implementing a university database system using SQL and ERD modeling.

The database includes:
- Students
- Instructors
- Departments
- Courses
- Sections
- Enrollments
- Classrooms
- Schedules
- Majors
- Payments

The project was developed using:
- SQL Database Design
- ER Diagram (ERD)
- Relational Schema
- Table Relationships
- Queries and Data Management
